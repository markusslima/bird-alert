Methods



.notify()

.notify({title: String, msg: String, className: String});

Method used to display the message

var birdAlert = new BirdAlert();
birdAlert.notify({
	msg: 'Mensage success!!!', 
	title: 'Success', 
	className: 'success'
});

The className parameter is used to set the css class of the theme. You can add new classes to the theme in css. The default class are: success, error, warning and info.

/* CSS */
.birdAlert .container-birdAlert.myClass {
	background: #dedede;
	color: #a0a0a0;
}

/* JavaScript */
birdAlert.notify({
	msg: 'Mensage bla bla bla!!!', 
	title: 'My Mensage', 
	className: 'myClass'
});

.set()

.set({option: value [, option: value]})

Updates the current object options

var birdAlert = new BirdAlert();
birdAlert.set({
	zIndex: 2100,
	clickToHide: false,
	showAnimation: 'fadeIn',
	autoHide: false
});

.setDefaults()

.set()

Returns all options to their default value

var birdAlert = new BirdAlert({
	clickToHide: false,
	showAnimation: 'fadeIn'
});
birdAlert.setDefaults();

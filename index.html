Requirements

Bootstrap 4 and jQuery.

Install

To use data attributes load the file at the end of the body
// Include
<script type="text/javascript" src="js/bootstrap-filestyle.min.js"> </script>

// Via JavaScript
$(":file").filestyle();

// Via data attributes
<input type="file" class="filestyle">
